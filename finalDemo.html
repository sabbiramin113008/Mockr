Authors API  ¶

This API renders the name of the authors to the associative books. It returns 20 authors ordered by Bnagla name. You can fetch any number of authors by using offset and limit parameters.

Authors API
GET/authors{?limit,offset,fields}

Example URI

GET http://api.rokomari.club/ecommerce/authors?limit=50&offset=10&fields=id,nameEnglish
URI Parameters
HideShow
offset
number (optional) Example: 10

Pagination offset ( 0, 10, any natural number )

limit
number (optional) Example: 50

Number of books to be fetched ( 10, 20, 30 or any natural number )

fields
string (optional) Example: id,nameEnglish

Comma separated input, the fields you want to get value ( id, nameBangla, nameEnglish )

Response  200
HideShow

Books API  ¶

This API returns 20 recent books. You can fetch any number of books by using offset and limit parameter.

Books API
GET/books{?author,publisher,category,sort,offset,limit,search,fields}

Example URI

GET http://api.rokomari.club/ecommerce/books?author=1&publisher=1&category=1&sort='+price'&offset=20&limit=30&search=Sabbir Amin&fields=author,category, created, detailBangla, detailEnglish, discount, edition,id,image,isbn,listPrice,nameBangla,nameEnglish,numberOfPages,publisher,rokomariID,salesPrice,status,updated
URI Parameters
HideShow
offset
number (optional) Example: 20

Pagination offset ( 0, 10 or any natural number )

limit
number (optional) Example: 30

Number of books to be fetched ( 10, 20, 30 or any natural number )

sort
stirng (optional) Example: '+price'

Ordering parameter. Here (+) means descending and (-) means ascending. In case of review and rating, it will return in descending order.

fields
string (optional) Example: author,category, created, detailBangla, detailEnglish, discount, edition,id,image,isbn,listPrice,nameBangla,nameEnglish,numberOfPages,publisher,rokomariID,salesPrice,status,updated

Comma separated input, the fields you want to get value.

author
string (optional) Example: 1

Filtering with author ( 1,2 , author id )

publisher
string (optional) Example: 1

Filtering with publisher ( 1 , 2, publisher id )

category
string (optional) Example: 1

Filtering with category ( 1, 2 , category id )

price<
string (optional) Example: 1000

Price Less than .

price>
string (optional) Example: 300

Price greater than .

discount<
string (optional) Example: 20

Discount less than .

discount>
string (optional) Example: 10

Discount greater than .

search
string (optional) Example: Sabbir Amin

Search via any string .

Search API  ¶

This API returns lists of books or authors or publishers according to search term. If you want to see only ebooks then set only the term parameter, if you want to see only ebooks then set filter= ebook or want to see publishers then set filter =publisher, for authors set filter=author and for categories set filter=category.

Search API
GET/search{?term,offset,limit,filter}

Example URI

GET http://api.rokomari.club/ecommerce/search?term=humayun&offset=10&limit=50&filter=author

HideShow
term
string (required) Example: humayun

Search string is used.

offset
string (optional) Example: 10

Pagination offset.

limit

Number of books to be fetched.

filter
string (optional) Example: author
